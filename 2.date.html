JavaScript Date Objects

1. Default Date Output

•Membuat objek tanggal: Objek tanggal dibuat menggunakan new Date()

Contoh

const date = new Date ["2022-03-25"];

•Parameter:Anda dapat memberikan parameter berupa string tanggal, angka (milidetik sejak 1 Januari 1970), atau format lainnya.

JavaScript Date Formats

JavaScript mendukung berbagai format string untuk membuat objek tanggal, seperti:

•ISO Date (International Standard): Format "YYYY-MM-DD" adalah format yang disarankan karena mendukung kompatibilitas penuh.

•Short Date: Format pendek seperti"MM/DD/YYYY" biasanya digunakan di Amerika Serikat.

•Long Date: Format panjang seperti"MMM DD YYYY"atau "25 Mar 2015" menggunakan nama bulan secara lengkap atau singkat.

Pentingnya format tanggal adalah untuk memastikan bahwa data tanggal diproses secara konsisten di berbagai browser dan zona waktu. Tanggal ISO adalah pilihan terbaik karena lebih andal untuk parsing dan kompatibilitas.

JavaScript Get Methods

JavaScript menyediakan metode untuk membaca komponen tanggal, seperti:

•getFullYear(): Mendapatkan tahun.

•getMonth(): Mendapatkan bulan (0-11, Januari dimulai dari 0).

•getDate(): Mendapatkan tanggal dalam bulan (1-31)

•getDay(): Mendapatkan hari dalam minggu (0-6, Minggu adalah 0).

Metode lainnya juga mencakup waktu seperti:

•getHours(), getMinutes(), getSeconds() dan getMilliseconds() untuk membaca komponen waktu.

• Metode getUTC...() digunakan untuk mendapatkan nilai berdasarkan waktu UTC.

JavaScript Set Methods

Selain membaca tanggal, JavaScript mendukung metode untuk mengatur nilai tanggal:

•setFullYear(year, month, day): Mengatur tahun, bulan, dan tanggal.

•setMonth(month, day): Mengatur bulan (0-11) dan opsional tanggal.

•setDate(day): Mengatur tanggal dalam bulan.

Anda juga dapat mengatur waktu dengan metode seperti getHours(), getMinutes(), getSeconds() dan getMilliseconds(). Semua metode ini memungkinkan manipulasi fleksibel atas nilai waktu dan tanggal dalam aplikasi Anda.
