JavaScript Random

1. Metode Dasar Math.random()

  • Deskripsi: Metode Math.random()menghasilkan angka acak antara 0 (inklusif) dan 1 (eksklusif).
Contoh

let randomNumber = Math.random();
console.log(randomNumber);     // Output: Angka antara 0 dan 1

2. Menghasilkan Angka Acak dalam Rentang Tertentu

  1. Antara 0 hingga 10:
    • Formula: Math.random() * 10
Contoh

let randomNumber = Math.random() * 10;
console.log(randomNumber);     // Output: Angka antara 0 dan 10
    2.Angka Bulat dalam Rentang Tertentu:
    • Menggunakan kombinasi Math.random dan Math.floor() :
Contoh

let randomInt = Math.floor(Math.random() * 10);
console.log(randomInt);     // Output: Angka bulat antara 0 dan 9
    3.Antara Nilai Minimum dan Maksimum:
    • Formula: Math.random() * 10
Contoh

let randomRange = Math.random() * (50 - 10) + 10;
console.log(randomRange);     // Output: Angka antara 10 dan 50
