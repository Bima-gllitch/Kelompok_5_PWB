JavaScript Booleans

Boolean adalah tipe data di JavaScript yang hanya memiliki dua nilai: true dan false. Boolean sering digunakan untuk mengontrol alur program dengan keputusan logis.

1. Boolean Sebagai Hasil Operasi Logis

Boolean dapat dihasilkan dari:

  • Perbandingan (misalnya 10 > 9 menghasilkan true)
  • Logika (operator seperti &&, ||, dan !).

    • 2. Tipe Data Boolean

    Boolean di JavaScript merupakan tipe data primitif. Boolean digunakan untuk mengevaluasi ekspresi logis dan dapat digunakan untuk pengujian kondisi dalam pernyataan if atau while.

    3. Konversi Ke Boolean

    JavaScript secara otomatis mengonversi beberapa nilai menjadi boolean:

  • Nilai falsy (seperti 0, "", null, undefined dan NaN) menjadi false
  • Nilai truthy (seperti angka non-0, string non-kosong, atau objek) menjadi true.

    • 4. Boolean Dalam Kondisi

    Nilai boolean sangat penting dalam struktur kendali seperti if, while, dan ternary operator.
