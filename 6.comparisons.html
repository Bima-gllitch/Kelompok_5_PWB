JavaScript Comparison and Logical Operators

1. Operator Perbandingan di JavaScript

Operator perbandingan digunakan untuk membandingkan nilai. Hasil dari operasi perbandingan adalah boolean: true atau false.

Operator Deskripsi Contoh
== Sama dengan 5 == '5' → true
=== Identik (sama nilai & tipe) 5 === '5' → false
!= Tidak sama dengan 5 != '5' → false
!== Tidak identik 5 !== '5' → true
> Lebih besar dari 5 > 3 → true
< Lebih kecil dari 5 < 3 → false
>= Lebih besar atau sama dengan 5 >= 5 → true
<= Lebih kecil atau sama dengan 5 <= 3 → false

2. Perbandingan dan Konversi Tipe

•Operator == membandingkan hanya nilai, sehingga 5 == '5' adalah true karena ada type coercion(konversi otomatis).

•Operator === membandingkan nilai dan tipe, sehingga lebih ketat. Contohnya, 5 === '5' adalah false karena tipe berbeda (angka dan string).

3. Contoh Penggunaan di JavaScript

Berikut beberapa implementasi umum

let x = 5;
console.log(x == "5");     // true
console.log(x === "5");     // false
console.log(x > 3);     // true
console.log(x <= 5);     // true
console.log(x !== 5);     // false

4. Perbandingan Nilai yang Tidak Biasa

a. Membandingkan objek atau array biasanya menghasilkan false, kecuali jika referensinya sama.

b. nul dan undefined memiliki perilaku spesifik saat dibandingkan:

  • null == undefined → true

    •null === undefined → false

5. Evaluasi Perbandingan

Evaluasi perbandingan mengikuti urutan prioritas.

Berikut beberapa implementasi umum

    console.log(5 > 3 > 1);     // false, karena (5 > 3) menghasilkan true, dan true > 1 adalah false.
