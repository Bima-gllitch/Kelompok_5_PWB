JavaScript if, else, and else if

1. If Statement

Digunakan untuk mengeksekusi blok kode hanya jika kondisi tertentu bernilai true.

2. If...Else Statement

Memungkinkan untuk menangani dua kasus:

    1. Blok kode pertama dijalankan jika kondisi bernilai true.
    2. Blok kedua dijalankan jika kondisi bernilai false.

3. If...Else If...Else Statement

    1. Digunakan untuk mengevaluasi beberapa kondisi secara berurutan.
    2. Jika salah satu kondisi terpenuhi, blok terkait akan dijalankan, dan proses evaluasi berhenti.
    3. Jika tidak ada kondisi yang terpenuhi, blok default di bawah else akan dijalankan.

4. Nested If Statements

Kondisi dapat ditumpuk dengan menempatkan if statement di dalam blok kode lainnya untuk memeriksa kondisi tambahan.

5. Ternary Operator

    1. Alternatif singkat untuk if...else yang digunakan untuk ekspresi sederhana.
    2. Menghasilkan nilai berdasarkan kondisi: satu untuk kondisi true dan lainnya untuk false.
