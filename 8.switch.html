JavaScript Switch Statement

1. Switch Statement

  • Switch digunakan untuk memilih salah satu dari banyak blok kode yang akan dijalankan berdasarkan nilai ekspresi.
  • Struktur switch menggantikan banyak if...else if dalam kasus di mana ada beberapa kondisi yang dievaluasi berdasarkan nilai.
  • Switch lebih rapi dan mudah dibaca dibandingkan banyak if...else if ketika memeriksa beberapa nilai.

2. Sintaks Dasar

    1. Expression yang dievaluasi di dalam switch
    2. Setiap case mewakili nilai yang mungkin dari ekspresi.
    3. break digunakan untuk menghentikan eksekusi setelah case tertentu cocok.
    4. default digunakan untuk menangani kasus ketika tidak ada nilai case yang cocok.

3. Fungsi Utama

    1. Eksekusi Kasus yang Cocok: Ketika nilai cocok dengan salah satu case, blok kode dalam kasus tersebut akan dijalankan.
    2. Pentingnya Break: Tanpa break, eksekusi akan terus berjalan ke case berikutnya (dikenal sebagai fall-through behavior).
    3. Default Case: Jika tidak ada nilai yang cocok, blok kode dalam default akan dieksekusi.

4. Fall-Through Behavior

    Fall-through dapat dimanfaatkan untuk menjalankan beberapa case bersama-sama. Jika tidak ada break, semua case berikutnya (termasuk default) akan dijalankan meskipun tidak cocok.
