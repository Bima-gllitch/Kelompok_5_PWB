JavaScript Loop For

1. For Loop

  • For loop digunakan untuk melakukan iterasi atau perulangan pada blok kode dengan jumlah iterasi yang telah ditentukan sebelumnya.
  • For loop sering digunakan ketika kita tahu berapa kali iterasi yang perlu dilakukan
Sintaks For Loop
    
    for (initialization; condition; increment) {
        // kode yang akan dijalankan
    }
    1. Initialization: Menetapkan nilai awal, yang biasanya digunakan untuk inisialisasi variabel loop.
    2. Condition: Mengevaluasi ekspresi boolean. Selama kondisi ini bernilai true, loop akan terus berjalan.
    3. Increment: Menentukan perubahan yang akan diterapkan pada variabel loop setelah setiap iterasi (biasanya penambahan atau pengurangan).

2. Cara Kerja For Loop

    1. Initialization akan dieksekusi sekali saat loop dimulai.
    2. Condition dievaluasi sebelum setiap iterasi. Jika true, kode dalam loop dieksekusi, dan increment diterapkan
    3. Jika condition menjadi false, loop akan berhenti dan melanjutkan ke kode setelah loop.

3. Penggunaan Umum

  • Digunakan untuk mengulang melalui array atau objek.
  • Dapat digunakan untuk operasi perhitungan atau manipulasi data dalam jumlah tetap.
Contoh
    
    for (let i = 0; i < 5; i++) {
        console.log(i);
    }
    loop akan mencetak angka dari 0 hingga 4.
