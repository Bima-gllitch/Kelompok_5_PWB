Rangkuman 2

Objek

Objek adalah kumpulan dari properti dan metode. Properti adalah nilai yang dimiliki objek, sedangkan metode adalah fungsi yang dapat dijalankan oleh objek.

Contoh:

const mobil = {
merek: "Toyota",
tahun: 2020,
info: function() {
  return `${this.merek} ${this.tahun}`;
}
};

Untuk menampilkan informasi objek:

console.log(mobil.info()); // Toyota 2020

Event

Event adalah aksi yang terjadi di dalam browser, seperti klik, hover, dan input. Kita bisa menangani event dengan menggunakan addEventListener.

Contoh:

document.getElementById("tombol").addEventListener("click", function() {
alert("Tombol diklik!");
});

String

String adalah tipe data yang digunakan untuk menyimpan teks. Kita dapat melakukan berbagai operasi pada string.

Contoh Metode String:

  • str.length - Menghitung panjang string.
  • str.toUpperCase() - Mengubah string menjadi huruf kapital.
  • str.slice() - Mengambil bagian dari string.

Contoh:

const kalimat = "Belajar JavaScript";
console.log(kalimat.length); // 17
console.log(kalimat.toUpperCase()); // BELAJAR JAVASCRIPT
console.log(kalimat.slice(0, 7)); // Belajar

Number

Number adalah tipe data yang digunakan untuk menyimpan angka. JavaScript juga mendukung BigInt untuk angka yang sangat besar.

Contoh:

const angka = 42;
const besar = 1234567890123456789012345678901234567890n;

Metode Number:

  • Number.isInteger() - Memeriksa apakah nilai adalah integer.
  • Number.parseFloat() - Mengonversi string menjadi float.

Contoh:

console.log(Number.isInteger(angka)); // true
console.log(Number.parseFloat("3.14")); // 3.14

Array

Array adalah tipe data yang digunakan untuk menyimpan kumpulan nilai. Kita dapat mengakses elemen array menggunakan indeks.

Contoh:

const buah = ["apel", "jeruk", "mangga"];
console.log(buah[0]); // apel

Metode Array:

  • array.push() - Menambahkan elemen ke akhir array.
  • array.pop() - Menghapus elemen terakhir dari array.
  • array.map() - Membuat array baru dengan hasil pemanggilan fungsi pada setiap elemen.

Contoh:

buah.push("pisang"); // ["apel", "jeruk", "mangga", "pisang"]
console.log(buah);
