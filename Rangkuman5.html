Rangkuman 5

1. Errors

Dalam JavaScript, terdapat beberapa jenis kesalahan (errors) yang mungkin muncul selama eksekusi kode.

  • SyntaxError: Kesalahan dalam tata bahasa kode.
  • ReferenceError: Variabel tidak dideklarasikan.
  • TypeError: Operasi pada tipe data yang tidak sesuai.

Gunakan tryâ€¦catch untuk menangani kesalahan ini.

2. Scope

Scope adalah area di dalam kode JavaScript di mana suatu variabel dapat diakses.

  • Global Scope: Variabel di luar fungsi atau blok.
  • Local Scope: Variabel dalam fungsi atau blok.

ES6 menambahkan block scope dengan let dan const.

3. Hoisting

Hoisting adalah mekanisme JavaScript yang memindahkan deklarasi variabel dan fungsi ke atas cakupannya saat eksekusi.

4. Strict Mode

Strict Mode mencegah penggunaan kode yang tidak aman. Aktifkan dengan menambahkan "use strict" di awal skrip.

5. this Keyword

this mengacu pada konteks objek saat diproses dalam fungsi dan berubah sesuai dengan cara fungsi dipanggil.

6. Arrow Functions

Arrow Functions adalah cara ringkas menuliskan fungsi dengan sintaks () => {}, di mana this merujuk ke konteks di luar fungsi tersebut.

7. Classes

ES6 memperkenalkan class sebagai kerangka untuk membuat objek dan mendukung pewarisan.

8. Modules

Modules memungkinkan pemisahan kode ke dalam beberapa file untuk struktur yang lebih teratur.

9. JSON

JSON adalah format data untuk pertukaran data antara server dan aplikasi web. Gunakan JSON.stringify() dan JSON.parse() untuk konversi data.

10. Debugging

Debugging adalah proses mencari dan memperbaiki kesalahan dalam kode menggunakan console.log() dan alat pengembang di browser.

11. JavaScript Style Guide

Style Guide menetapkan aturan untuk penulisan kode yang konsisten dan mudah dibaca.

12. Best Practices

Beberapa praktik terbaik dalam JavaScript termasuk penggunaan const untuk variabel konstan dan memisahkan logika dalam fungsi kecil.

13. Mistakes

Kesalahan umum seperti menggunakan == alih-alih === atau lupa mendeklarasikan variabel.

14. Performance

Optimalkan kinerja dengan mengurangi akses DOM dan menggunakan teknik debouncing atau throttling pada event yang sering dipicu.

15. Reserved Words

Reserved words adalah kata-kata yang tidak dapat digunakan sebagai nama variabel, seperti break dan class.
